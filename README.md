
# English - Lingala - Kikongo - French Dataset 🇨🇩

This repository contains a multilingual dataset for:

- Lingala
- Kikongo (Great Bandundu)
- French

## 📊 Description
This dataset provides parallel sentences aligned with English to support:

- Natural Language Processing (NLP)
- Machine Translation
- AI and language technology development
- Digital inclusion of under-resourced African languages

## 📁 Dataset
- File: `English - Lingala - Kikongo - French dataset`
- Format: CSV
- Languages: 5 columns (English + 3 Congolese languages)

## 🎯 Goal
To improve the presence of Congolese languages in digital systems and AI technologies.
